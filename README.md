Open Source packages from Triângulos Tecnologia.

## Contributing

For contributing, building `config` is required, as this set some settings accross all the repository.

1. Fork the repository
2. Clone it
3. Build `config`

```sh
pnpm build:config
```

4. Contribute ;)
