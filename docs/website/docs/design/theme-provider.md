---
title: ttoss Theme Provider
---

The Theme Provider is the React integration layer for [`@ttoss/fsl-theme`](/docs/design/design-system/design-tokens/model). It manages theme switching, mode resolution (light / dark / system), SSR flash prevention, and exposes semantic tokens to your component tree.

For setup and usage — `ThemeProvider`, `ThemeScript`, `ThemeHead`, `ThemeStyles`, `useColorMode`, `useTokens`, `useResolvedTokens`, and the Next.js / SSR patterns — see the package [README](https://github.com/ttoss/ttoss/blob/main/packages/fsl-theme/README.md).
