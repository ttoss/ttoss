# Chakra UI Integration

ttoss seamlessly integrates with [Chakra UI](https://chakra-ui.com/), providing a powerful component library with consistent theming through custom recipes and semantic color tokens.

## What is Chakra UI?

Chakra UI is a modern React component library that provides:

- **Accessible components** following WAI-ARIA standards
- **Composable design system** with component primitives
- **Theming capabilities** through recipes and design tokens
- **TypeScript support** with full type safety
- **Dark mode support** out of the box

## How ttoss Uses Chakra UI

The `@ttoss/ui` package extends Chakra UI with custom theming through:

### 1. Semantic Color Tokens

ttoss provides semantic color tokens organized by purpose:

- **navigation**: Navigation elements (menus, headers, sidebars)
- **display**: Main content (text, backgrounds, borders)
- **action**: Interactive elements (buttons, links, CTAs)
- **input**: Form inputs and controls
- **feedback**: Status messages (success, error, warning, info)

### 2. Custom Recipes

Recipes define color theming for components while using Chakra's default tokens for spacing, sizing, and typography:

- **Recipes** (`defineRecipe`): For single-part components like Button
- **Slot Recipes** (`defineSlotRecipe`): For multi-part components like Steps

### 3. Theme Provider Integration

The `ThemeProvider` from `@ttoss/ui` automatically applies your theme to all Chakra UI components:

```tsx
import { ThemeProvider } from '@ttoss/ui';
import { BruttalTheme } from '@ttoss/theme/Bruttal';
import { Button } from '@chakra-ui/react';

export const App = () => (
  <ThemeProvider theme={BruttalTheme}>
    {/* Button automatically styled with BruttalTheme */}
    <Button>Click me</Button>
  </ThemeProvider>
);
```

## Using Chakra UI Components

Import components directly from `@chakra-ui/react`:

```tsx
import { Button, Box, Flex, Text, Stack } from '@chakra-ui/react';

export const Example = () => (
  <Box p={4}>
    <Flex gap={4} direction="column">
      <Text fontSize="xl" fontWeight="bold">
        Welcome
      </Text>
      <Button>Get Started</Button>
    </Flex>
  </Box>
);
```

## Custom Theming

### Adding Custom Recipes

Create custom recipes in your theme configuration:

```tsx
import { defineRecipe, defineSlotRecipe } from '@chakra-ui/react';
import type { Theme } from '@ttoss/ui';

export const customTheme: Theme = {
  colors: {
    primary: '#007acc',
  },
  recipes: {
    button: defineRecipe({
      base: {
        color: 'action.text.primary.default',
        backgroundColor: 'action.background.primary.default',
      },
    }),
  },
  slotRecipes: {
    steps: defineSlotRecipe({
      slots: ['root', 'item', 'indicator'],
      base: {
        indicator: {
          color: 'display.text.accent',
        },
      },
    }),
  },
};
```

### Recipe Development Principles

When creating or modifying Chakra UI recipes:

- **✅ Use ttoss semantic color tokens** for all color-related properties
- **✅ Use Chakra default tokens** for spacing, sizing, typography, borders, etc.
- **❌ Do not include** non-color design tokens in recipes
- **❌ Do not include** layout properties in recipes

This approach ensures:

- Consistent theming across applications
- Compatibility with Chakra's design system
- Minimal maintenance burden
