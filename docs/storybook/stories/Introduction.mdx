# ttoss Components Storybook

Welcome to the **ttoss ecosystem components library**! This Storybook showcases all React components available in `@ttoss/components`.

## Quick Start

```shell
pnpm add @ttoss/components @ttoss/ui @emotion/react @ttoss/react-hooks
```

## What's Inside

**16 Production-Ready Components** that integrate seamlessly with the ttoss ecosystem:

### Layout & Navigation

- **Accordion** - Collapsible content sections with accessibility
- **Drawer** - Slide-out panels from screen edges
- **Menu** - Dropdown menus with customizable triggers
- **Modal** - Theme-aware modals with accessibility features
- **Tabs** - Tab navigation with content panels

### Content & Display

- **List** - Unordered lists with customizable items
- **Markdown** - Render markdown with theme integration
- **Table** - Flexible tables with sorting/pagination (TanStack Table)
- **JsonEditor** - JSON editing with syntax highlighting
- **JsonView** - JSON viewer component

### Form & Input

- **Search** - Debounced search with loading states
- **FileUploader** - Drag-and-drop file upload with progress

### Notifications

- **NotificationCard** - Display notification messages
- **NotificationsMenu** - Menu for displaying notifications
- **Toast** - Toast notification system

### Utilities

- **InstallPwa** - PWA installation prompt

## Theme Integration

All components are **theme-aware** and integrate with:

- `@ttoss/ui` design system
- Emotion CSS-in-JS
- Theme-UI tokens and responsive values

## How to Use This Storybook

1. **Browse Components** - Navigate through the sidebar to explore each component
2. **Interactive Examples** - Try out live examples in the Canvas tab
3. **View Documentation** - Check the Docs tab for usage patterns and API
4. **Copy Code** - Use the code examples directly in your projects
