# Plano: dotDensity e proportionalCircles no Geovis

## Visão Geral

Adicionar dois novos `mapType` que geram automaticamente layers `point` (geometry) com paint e configurações apropriadas, seguindo o padrão existente do `choropleth`.

- **`dotDensity`**: Círculos pequenos com paint fixo para visualização de densidade
- **`proportionalCircles`**: Círculos proporcionais com `sizeBy` para `circleRadius` data-driven

Ambos são **mapTypes** (não geometry types) — o geometry type resultante é sempre `point`.

---

## Fase 1: `dotDensity` (mapType)

Novo mapType que renderiza pontos como círculos pequenos com defaults ideais para visualização de densidade.

### Defaults de paint

```typescript
const DOT_DENSITY_PAINT = {
  circleColor: '#E4572E',
  circleRadius: 2.4,
  circleStrokeColor: '#FAF9F7',
  circleStrokeWidth: 0.5,
};
```

### Spec resultante (após resolução)

```json
{
  "mapType": "dotDensity",
  "sources": [{ "id": "my-points", "type": "geojson", "data": "..." }],
  "layers": [
    {
      "id": "my-points-dots",
      "sourceId": "my-points",
      "geometry": "point",
      "mapDataId": "events",
      "paint": {
        "circleColor": "#E4572E",
        "circleRadius": 2.4,
        "circleStrokeColor": "#FAF9F7",
        "circleStrokeWidth": 0.5
      }
    }
  ]
}
```

### Passos

1. **Tipos**: Adicionar `'dotDensity'` ao `MapType` em `types.ts:91`
2. **Schema**: Adicionar `'dotDensity'` ao enum `mapType` em `schema.json:24`
3. **Resolver**: Criar `src/spec/mapTypeDefaults/dotDensity.ts`
   - Função `resolveDotDensity(spec, sourceId, mapDataEntry)`
   - Retorna `layers: [pointLayer]` com `geometry: 'point'` e paint defaults
   - `mapDataId` do primeiro `mapData` entry
4. **Dispatcher**: Atualizar `resolveSpecFromMapType` em `mapTypeDefaults.ts:79`
   - Adicionar branch `if (spec.mapType === 'dotDensity' && firstMapData)`
   - Chamar `resolveDotDensity`
5. **Validação**: Verificar que `dotDensity` requer source geojson (já coberto pelo dispatcher)
6. **Fixture**: Criar fixture de exemplo em `src/fixtures/`
7. **Testes**: Testar resolver e integração

## Fase 2: `proportionalCircles` (mapType + sizeBy)

Novo mapType que usa `sizeBy` para `circleRadius` data-driven, gerando círculos proporcionais aos valores dos dados.

### Nova interface `SizeBy`

```typescript
export interface SizeBy {
  /** Range de pixels [min, max] para o circleRadius. */
  range: [number, number];
  /** Transformação aplicada antes do mapeamento. Default: 'linear'. */
  transform?: 'linear' | 'sqrt' | 'log';
}
```

Adicionar à `VisualizationLayer`:

```typescript
export interface VisualizationLayer {
  // ...existing fields...
  sizeBy?: SizeBy;
}
```

### Spec de entrada (simplificado)

```json
{
  "mapType": "proportionalCircles",
  "sources": [{ "id": "my-points", "type": "geojson", "data": "..." }],
  "mapData": [{ "mapDataId": "population", "mapId": "my-points", "data": [...] }]
}
```

### Spec resultante (após resolução)

```json
{
  "mapType": "proportionalCircles",
  "sources": [{ "id": "my-points", "type": "geojson", "data": "..." }],
  "layers": [
    {
      "id": "my-points-circles",
      "sourceId": "my-points",
      "geometry": "point",
      "mapDataId": "population",
      "sizeBy": { "range": [3, 20], "transform": "sqrt" },
      "paint": {
        "circleColor": "#3b82f6",
        "circleOpacity": 0.7,
        "circleStrokeColor": "#ffffff",
        "circleStrokeWidth": 1
      }
    }
  ],
  "legends": [
    {
      "id": "population-legend",
      "title": "population",
      "sizeBy": { "range": [3, 20], "transform": "sqrt" }
    }
  ]
}
```

### Passos

1. **Tipos**: Adicionar `'proportionalCircles'` ao `MapType` em `types.ts:91`
2. **Schema**: Adicionar `'proportionalCircles'` ao enum `mapType` em `schema.json:24`
3. **Tipos**: Criar interface `SizeBy` e adicionar `sizeBy?` ao `VisualizationLayer` em `types.ts`
4. **Schema**: Adicionar `sizeBy` ao schema de layer em `schema.json`
5. **Resolver**: Criar `src/spec/mapTypeDefaults/proportionalCircles.ts`
   - Função `resolveProportionalCircles(spec, sourceId, mapDataEntry)`
   - Retorna `layers: [pointLayer]` com `geometry: 'point'`, paint defaults e `sizeBy`
   - Gera `legends` com `sizeBy` para o componente de legenda
6. **Dispatcher**: Atualizar `resolveSpecFromMapType` em `mapTypeDefaults.ts:79`
   - Adicionar branch `if (spec.mapType === 'proportionalCircles' && firstMapData)`
   - Chamar `resolveProportionalCircles`
7. **Adapter**: Atualizar `layerTranslation.ts` para suportar `sizeBy`
   - No `buildPoint`: quando `layer.sizeBy` estiver presente, gerar expressão MapLibre `interpolate` para `circle-radius`
   - Usar `mapData` para obter min/max dos valores
8. **Legend**: Atualizar `legendTranslation.ts` para suportar legenda de `sizeBy`
9. **Fixture**: Criar fixture de exemplo em `src/fixtures/`
10. **Testes**: Testar resolver, adapter e integração
