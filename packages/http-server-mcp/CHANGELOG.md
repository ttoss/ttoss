## 0.1.0

### Minor Changes

- Initial release of MCP (Model Context Protocol) server integration for @ttoss/http-server
- Added `createMcpRouter` function to create Koa routers for MCP servers
- Support for custom endpoint paths and session management
- Full integration with @modelcontextprotocol/sdk
- Comprehensive test coverage
