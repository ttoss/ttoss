# @ttoss/fsl-ui

React component library built on React Aria Components, styled with
`@ttoss/fsl-theme` design tokens.

This is the A/B "bare" condition: the model receives only what it could
discover without documentation — the package names and the export list
(what IDE autocomplete would show). No usage rules, no semantic model.

## Setup

```
pnpm add @ttoss/fsl-ui @ttoss/fsl-theme react react-dom
```

Theme utilities: `createTheme` from `@ttoss/fsl-theme`; `ThemeProvider` from
`@ttoss/fsl-theme/react`.
